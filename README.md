# WhatsApp Automation Bot

Este projeto tem como objetivo desenvolver uma solução completa de automação de atendimento ao cliente via WhatsApp, utilizando Python e inteligência artificial.

## Funcionalidades
- Atendimento automático 24/7 via WhatsApp.
- Fluxo de mensagens com saudações e dúvidas frequentes.
- Chatbot inteligente para responder perguntas complexas.
- Encaminhamento para atendente humano quando necessário.
- Registro de histórico de conversas.
- Análise de dados para melhoria contínua.

## Tecnologias
- Python 3
- API do WhatsApp Business / Twilio
- Banco de dados SQLite
- Bibliotecas de IA (Transformers, NLTK)
